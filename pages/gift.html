Giftigt for hunden

Ligesom med alle andre dyr og mennesker er der en masse som hunde ikke kan tåle i større mængder og noget er meget giftigt. Menneskemedicin, bagepulver, rå kartofler, ukogte pasta samt ris og tomater er alt sammen noget hunden absolut ikke kan tåle.

For at få etoverblik har vi samlet en liste over en masse råvarer som man skal forsøge at undgå eller absolut ikke skal servere sin hund samt den potentielle virkning af disse produkter.

Giftigt Virkning
Alkohol Forgiftning, koma eller sommetider endda døden
Avocado (både planten og frugten Opkast og diarre
Ben fra fjerkræ eller fisk Forstoppelse eller sønderrivning af hundens fordøjelsessystem (dette gælder især kogte ben, også fra svin, okse og lam)
Chokolade, kaffe, te og koffein Opkast og diarre samt påvirkning af hjertet og nervesystemet
Citrusolie (især ekstrakt) Opkast
Fedtafskæringer af kød Bugspytkirtelbetændelse
Fisk (rå, konverseret og tilberedt) Hvis hunden udelukkende fodres med fisk, kan dette resultere i nedsat appetit, anfald og endda døden (fisk i mindre mængder kan med fordel indgå i kosten)
Fordærvede eller rådne fødevarer og affald Opkast og diarre samt påvirkning af enkelte organer
Gærdej Kan øge mængden af tarmluft i fordøjelsessystemet, hvilket kan medføre smerter og endda skader på maven eller tarmene
Humle Åndedrætsbesvær, høj hjerterytme, høj kropstemperatur, anfald og døden
Jerntilskud til mennesker Skader på fordøjelsessystemet og nogle organer heriblandt leveren og nyrerne
Macadamianødder Påvirker fordøjelsessystemet, nervesystemet og muskulaturen
Majskolber Selvom hunden godt kan spise og fordøje majs, kan majskolben (eller dele af den) sætte sig fast i hundens fordøjelsessystem og skabe en blokering
Mælk og mælkeprodukter Diarre (hunde kan normalt godt tåle mindre mængder og forgiftes kun ved større mængder)
Løg og hvidløg (rå, tilberedte og pulver) Skader de røde blodlegemer, hvilket kan føre til anæmi
Kattemad Opkast og diarre (det indeholder for meget protein og fedt)
Kerner af kakifrugter Blokering af tarmsystemet og tarmbetændelse
Rå æg Hunde må gerne fodres med rå æg, men rå æg kan medføre problemer med pels og hud samt risiko for forgiftning med salmonellabakterier
Råt kød Hunde må gerne fodres med råt kød, men det kan indeholde salmonella og E. coli-bakterier, hvilket kan medføre opkast og diarre
Rabarberblade Kan påvirke fordøjelses-, nerve- og urinvejssystemet
Salt Kan føre til ubalance i salt- og væskebalance, hvis det indtages i større mængder
Sukkerholdige fødevarer Kan medføre overvægt, tandproblemer og sukkersyge
Svampe Påvirkning af forskellige kropssystemer, hvilket kan resultere i shock og endda døden
Rester (fra aftensmad og lignende) Rester fra menneskets måltider er ernæringsmæssigt ude af balance i forhold til hundens behov – resterne bør derfor maksimalt udgøre 10% af hundens kost
Tobak Skader fordøjelses- og nervesystemet, og kan medføre høj hjerterytme, kollaps, koma og døden
Sten fra ferskner og blommer Blokering af fordøjelsessystemet
Sødemiddel Kan medføre lavt blodsukker, hvilket kan føre til opkast, kollaps og generel svækkelse – i store mængder kan det forårsage leverskader
